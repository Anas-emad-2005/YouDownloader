# YouDownloader 🚀

[العربية]
أداة ويب احترافية وبسيطة لتحميل مقاطع الفيديو من يوتيوب بجودات مختلفة، تم تطويرها باستخدام Flask و yt-dlp. المشروع مصمم ليكون سريعاً وسهل الاستخدام مع واجهة عصرية.

[English]
A professional and simple web tool to download YouTube videos in various qualities, developed using Flask and yt-dlp. The project is designed to be fast and user-friendly with a modern interface.

---

## المميزات | Features ✨

* **Multi-Quality Support:** اختيار جودات متنوعة (من 144p إلى أعلى جودة متاحة).
* **Audio Download:** إمكانية تنزيل الصوت فقط بصيغة MP3/M4A.
* **Auto-Naming:** تسمية الملف تلقائياً باسم الفيديو الأصلي.
* **Modern UI:** واجهة مستخدم متجاوبة مع الوضع الداكن (Dark Mode).
* **Mobile Friendly:** يعمل بكفاءة على أجهزة الكمبيوتر والهواتف الذكية.

---

## تقنيات المشروع | Tech Stack 🛠️

* **Backend:** Python 3.10+, Flask
* **Scraper Library:** yt-dlp
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Deployment:** Render / PythonAnywhere

---

## كيف يعمل؟ | How it Works? 📖

1. ضع رابط الفيديو في الحقل المخصص.
2. اضغط على "فحص الرابط" لجلب البيانات والجودات.
3. اختر الجودة المطلوبة ونوع الملف (فيديو/صوت).
4. اضغط على "بدء التحميل" وسيبدأ التنزيل مباشرة في جهازك.
